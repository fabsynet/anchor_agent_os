  <action>
1. **MetricCard** (metric-card.tsx): Reusable summary card component.
   - Props: title, value (number), icon (LucideIcon), trend? (percentage change), subtitle?
   - Styling: slate-800 background, white text, icon with blue-600 accent, large bold value, smaller title below
   - Use lucide-react icons: Building2 (agencies), Users (users), FileText (policies), UserCheck (clients)

2. **HealthAlerts** (health-alerts.tsx): Alert list component.
   - Props: alerts: HealthAlert[]
   - Renders each alert as a card with severity-based styling (warning=amber, critical=red)
   - Shows type icon, message, value vs threshold
   - Empty state: green "All systems healthy" message
   - Renders inside a "System Health" section header

3. **TimeRangeSelector** (time-range-selector.tsx): Preset + custom date picker.
   - Props: onChange: (params: { startDate: string, endDate: string }) => void
   - Preset buttons: 7d, 30d, 90d, All Time (styled as toggle group)
   - Custom date picker: two date inputs (start/end) that appear when "Custom" is selected
   - **Date mapping logic** (converts presets to startDate/endDate for the growth API):
     - 7d: startDate = 7 days ago (ISO), endDate = now (ISO)
     - 30d: startDate = 30 days ago, endDate = now
     - 90d: startDate = 90 days ago, endDate = now
     - All Time: omit both params (API defaults to last 12 months)
     - Custom: use selected start/end dates
   - onChange always emits { startDate, endDate } as ISO strings — the growth endpoint at GET /admin/metrics/growth accepts these directly as query params
   - Use date-fns subDays/formatISO for date calculations

4. **Export utils** (export-utils.ts):
   - `exportToCSV(data: Record<string, any>[], filename: string)`: Uses PapaParse unparse(). Dynamic import for bundle size.
   - `exportToPDF(title: string, data: Record<string, any>[], columns: string[])`: Uses jsPDF + jspdf-autotable. Dynamic import.
   - Follow the exact pattern from apps/web export utilities.

Install PapaParse and jsPDF in apps/admin if not already present:
```bash
cd apps/admin && pnpm add papaparse jspdf jspdf-autotable && pnpm add -D @types/papaparse
```
